# Travel Website

This project is a multi-page travel booking website prototype built with HTML and CSS.

## Pages
- Homepage
- Search Results
- Checkout Page
- Trips
- Help Center

## Project Structure
Each page has its own folder with:
- `index.html`
- `style.css`
- `images/`

## How to Run
1. Clone the repository
2. Open the project folder in VS Code
3. Open any page's `index.html` file in your browser

Example:
- `Homepage/index.html`
- `SearchResults/index.html`

## Notes
- Images may be placeholders during development
- Styling is currently page-specific
- Navigation links connect pages using relative file paths

## Team Workflow
- Build one page at a time
- Commit changes by page
- Test links between pages after each update
